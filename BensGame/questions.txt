1. What does HTML stand for?

A) Hyper Trainer Marking Language
B) Hyper Text Markup Language
C) Hyperlinking Text Marking Language
D) High Tech Modern Language

Answer: B) Hyper Text Markup Language
2. What is the correct way to create a hyperlink in HTML?

A) <a>www.example.com</a>
B) <a href="www.example.com">Click Here</a>
C) <link url="www.example.com">Click Here</link>
D) <hyper>Click Here</hyper>

Answer: B) <a href="www.example.com">Click Here</a>
3. Which HTML tag is used for the largest heading?

A) <h1>
B) <heading>
C) <h6>
D) <head>

Answer: A) <h1>
4. How do you insert an image in HTML?

A) <img src="image.jpg">
B) <image>image.jpg</image>
C) <img>image.jpg</img>
D) <picture src="image.jpg">

Answer: A) <img src="image.jpg">
5. Which tag is used to define a paragraph?

A) <para>
B) <p>
C) <pg>
D) <paragraph>

Answer: B) <p>
6. Which HTML element is used to create an unordered list?

A) <list>
B) <ul>
C) <ol>
D) <li>

Answer: B) <ul>
7. What is the purpose of the <title> tag?

A) It defines the page title in the browser tab
B) It makes text bold
C) It creates a section heading
D) It underlines text

Answer: A) It defines the page title in the browser tab
8. How do you create a line break in HTML?

A) <lb>
B) <br>
C) <break>
D) <hr>

Answer: B) <br>
9. What is the correct HTML tag for inserting a horizontal line?

A) <line>
B) <hr>
C) <hl>
D) <br>

Answer: B) <hr>
10. Which HTML attribute specifies an alternative text for an image?

A) title
B) alt
C) src
D) description

Answer: B) alt
11. What is the correct way to create a comment in HTML?

A) <!-- This is a comment -->
B) // This is a comment
C) /* This is a comment */
D) ** This is a comment **

Answer: A) <!-- This is a comment -->
12. What does the <body> tag contain?

A) Metadata about the page
B) The visible content of the webpage
C) External CSS files
D) JavaScript functions only

Answer: B) The visible content of the webpage
13. Which of the following tags is used to create a table?

A) <tbl>
B) <tr>
C) <td>
D) <table>

Answer: D) <table>
14. Which HTML tag is used to define a form for user input?

A) <input>
B) <form>
C) <text>
D) <fieldset>

Answer: B) <form>
15. What does the <meta> tag do?

A) It defines metadata about the document
B) It creates a footer section
C) It adds a logo to the webpage
D) It formats text

Answer: A) It defines metadata about the document
16. What is the purpose of the <label> element?

A) To style text
B) To create a clickable text for form inputs
C) To add metadata
D) To create a title

Answer: B) To create a clickable text for form inputs
17. Which input type is used for a password field?

A) <input type="text">
B) <input type="password">
C) <input type="secure">
D) <input type="hidden">

Answer: B) <input type="password">
18. How do you create a dropdown list in HTML?

A) <dropdown>
B) <list>
C) <select>
D) <option>

Answer: C) <select>
19. What tag is used to define an inline frame (iframe)?

A) <frame>
B) <iframe>
C) <window>
D) <embed>

Answer: B) <iframe>
20. What is the correct way to specify that a link should open in a new tab?

A) <a href="url" new>
B) <a href="url" target="_blank">
C) <a href="url" target="new-tab">
D) <a href="url" window="_new">

Answer: B) <a href="url" target="_blank">
